# Shop2Pay

### Install
First you need install Node.js (version 8.x is recommended) and Meteor framework.

- [Node.js](https://nodejs.org/)
- [Meteor](https://www.meteor.com/)

Install and run

```sh
npm install
npm start
```

You also can testing by

```sh
npm test
```

And open `http://localhost:3000/` in your browser.
